   self.model = self.model.to('cpu')

   self.model = self.model.to(original_device)

## 📚 技术细节

### NDCG@5计算方法

```python
def compute_ndcg_at_5(predictions):
    """
    NDCG (Normalized Discounted Cumulative Gain) 是排序评估的关键指标
    
    计算公式:
    DCG_5 = Σ(2^rel_i - 1) / log_2(i+1),  i从1到5
    NDCG_5 = DCG_5 / IDCG_5
    
    其中:
    - rel_i: 第i名的相关性分数（0/1）
    - IDCG_5: 理想排序的DCG值
    - 分母log_2(i+1)进行位置折减，越靠前的排序权重越大
    """
```

### F1-Score vs NDCG的区别

| 指标 | 用途 | 特性 |
|------|------|------|
| **F1-Score** | 二元分类评估 | 不考虑排序位置，只看正确性 |
| **NDCG@5** | 排序质量评估 | 考虑排序位置，高相关性必须在前5名 |

两者结合能全面评估模型的分类和排序能力。
